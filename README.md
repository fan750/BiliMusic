# BiliMusic

一个bilibili音乐播放器

## 功能

* [X] 营销落地页
* [X] 登录/注册页面原型
* [X] 本地歌单创建
* [X] 营销落地页
* [X] 登录 / 注册界面原型
* [X] 本地歌单创建
* [X] 本地歌单列表展示
* [X] 歌单详情查看
* [X] 手动添加歌曲（通过输入 BV 号）
* [X] 从歌单中删除歌曲
* [X] 基于 **Bilibili 嵌入播放器**的单曲播放
* [X] 基础的下一首自动切换逻辑
* [ ] 真实用户登陆注册
* [ ] 歌单重命名与删除
* [ ] BV自动解析标题、封面、时长
* [ ] 播放全部
* [ ] 上一首/下一首
* [ ] 暂停/继续播放
* [ ] Bilibili搜索功能
* [ ] 推荐页
* [ ] 更完善的播放器控制
* [ ] 本地存储层（打包的话最好做SQLite）
* [ ] Electron桌面端接入

## 快速开始

### 进入项目目录
```bash
cd BiliMusic
```
### 安装依赖
```bash
npm install
```
### 启动开发环境
```bash
npm run dev
```
